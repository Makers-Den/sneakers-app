# Sneakers App

## Running the app locally

1. Setup [iOS Simulator](https://docs.expo.dev/workflow/ios-simulator/) or [Android Studio Emulator](https://docs.expo.dev/workflow/android-studio-emulator/)

2. Install dependencies:

```sh
yarn install
```

3. Start the app:

```sh
yarn start:ios
```

or

```sh
yarn start:android
```
